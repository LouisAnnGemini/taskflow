# TaskFlow 功能说明书

TaskFlow 是一款高效、直观的任务管理系统，旨在帮助个人和团队更好地组织、跟踪和协作完成各项任务。

## 核心功能

### 1. 任务看板 (Kanban)
- **多列管理**：默认包含待办、进行中、审核中、已完成、延期等状态列。
- **拖拽操作**：支持通过拖拽任务卡片快速变更任务状态。
- **直观展示**：卡片显示任务标题、优先级、负责人、截止日期、进度条及媒介标签。
- **栏目隐藏**：支持在看板视图中一键显示或隐藏特定栏目，隐藏后剩余栏目自动居中。
- **全屏布局**：看板视图优化为铺满整个屏幕，提供更广阔的视觉范围。

### 2. 仪表盘 (Dashboard)
- **置顶优先级**：快速查看被标记为“置顶”的关键任务。
- **今日及逾期**：聚焦于今天需要完成或已经逾期的任务。
- **等待我审核**：专门列出需要当前用户进行审核的任务。
- **委派给他人**：跟踪由我创建并委派给其他成员的任务。

### 3. 时间线 (Timeline)
- **甘特图视图**：基于任务的开始日期和截止日期，在时间轴上可视化展示任务排期。
- **进度展示**：在时间轴卡片上直接显示任务完成百分比。
- **快速跳转**：点击时间线上的任务可直接打开详情。
- **筛选功能**：支持按状态、优先级、负责人、委派状态、媒介标签、时间段及进度范围进行筛选，点击“查找”后显示结果，避免界面过长。

### 4. 备忘录 (Memos)
- **独立记录**：提供单独的备忘录标签页，用于记录与任务无关的想法、灵感或笔记。
- **CRUD 操作**：支持备忘录的创建、修改、删除，并支持 inline 确认删除操作。

### 5. 搜索与筛选 (Search & Filter)
- **全局搜索**：支持按标题和描述进行关键词搜索。
- **排序功能**：搜索结果自动按任务更新时间 (`updatedAt`) 从新到旧排列。
- **多维筛选**：
    - 按状态、优先级筛选（支持“排除”选项）。
    - 按创建人、负责人、提出人（审核人）筛选（支持“排除”选项）。
    - 按委派状态（已委派/未委派）筛选（支持“排除”选项）。
    - 按媒介标签（微信、邮件、OA等）筛选（支持“排除”选项）。

### 6. 任务详情与编辑 (Task Details)
- **标签页布局**：任务详情页面采用标签页形式，分为“详情”、“活动日志”和“关联图谱”。
- **详情页**：包含任务标题、流程可视化、描述、子任务、关联任务以及任务字段侧边栏。
- **活动日志页**：专门展示活动日志，支持日志的查看与管理（删除、编辑）。
- **关联图谱页**：可视化展示任务间的关联关系，支持节点点击跳转、滚轮缩放及拖拽平移。
- **流程可视化**：直观展示任务从“待办”到“完成”的流转路径，特别标注委派状态。当任务处于“已完成”状态时，最后一个“完成”节点显示为绿色的已勾选节点。
- **子任务管理**：支持创建、勾选、修改子任务标题，细化工作内容。支持快速删除子任务。
- **关联任务**：支持任务间的双向关联，点击关联任务可直接跳转至目标任务详情。
- **侧边栏属性**：快速修改负责人、委派对象、审核人、优先级、日期、进度及标签。
- **优化人员选择**：负责人和委派对象字段优化，固定“我”在首位，其他人员隐藏并支持搜索选择。
- **删除确认**：所有删除操作（任务、备忘录、设置项）均采用页面内确认框，提升可见性。

### 7. 任务卡片展示 (Task Card)
- **视觉区分**：
    - **置顶任务**：显示红色边框和浅红色背景。
    - **委派任务**：显示蓝色边框和浅蓝色背景，并在右下角显示紫色委派标记。
- **简洁操作**：移除了卡片上的委派功能按钮，委派操作仅在任务详情页进行。

### 8. 快速捕获 (Quick Capture)
- **即时创建**：在页面顶部提供输入框，支持快速添加新任务，样式与各标签页保持一致。
- **立即完成**：支持“立即完成”模式，适用于那些随手完成并需要记录的小事。

### 9. 数据管理 (Data Management)
- **实时持久化**：所有操作实时保存至浏览器的 Local Storage，刷新页面不丢失数据。
- **导出数据**：支持将所有任务及设置导出为 JSON 文件进行备份。
- **导出 Excel**：支持将任务列表导出为 Excel 文件。
- **导入数据**：支持从 JSON 文件恢复数据，导入前提供覆盖确认提示，并显示操作反馈。

### 10. 系统设置 (Settings)
- **团队管理**：自定义团队成员名单。
- **看板定制**：支持添加、修改或删除看板列，自定义列颜色和图标。
- **优先级定制**：自定义任务优先级及其对应的视觉样式。
- **媒介标签定制**：自定义任务来源或沟通渠道标签（如：微信、钉钉、邮件等）。
- **自定义字段**：支持添加文本、数字、日期、单选、多选字段，支持字段排序、显示隐藏，以及选项重排序。当选项超过10个时，自动转为搜索选择框。

## 技术特性
- **响应式设计**：适配桌面端和移动端。
- **图标系统**：集成 Lucide React 图标库，支持 Emoji 自定义图标。
- **状态管理**：基于 Zustand 实现高效的前端状态管理。
- **持久化方案**：使用 Zustand Persist 中间件。
