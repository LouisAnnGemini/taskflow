# TaskFlow 功能说明书

TaskFlow 是一款高效、直观的任务管理系统，旨在帮助个人和团队更好地组织、跟踪和协作完成各项任务。

## 核心功能

### 1. 任务看板 (Kanban)
- **多列管理**：默认包含待办、进行中、审核中、已完成、延期等状态列。
- **拖拽操作**：支持通过拖拽任务卡片快速变更任务状态。
- **直观展示**：卡片显示任务标题、优先级、负责人、截止日期、进度条及媒介标签。
- **栏目隐藏**：支持在看板视图中一键显示或隐藏特定栏目，隐藏后剩余栏目自动居中。
- **全屏布局**：看板视图优化为铺满整个屏幕，提供更广阔的视觉范围。
- **移动端适配**：在手机端，看板列宽自动调整为屏幕宽度的 85%，并引入滑动对齐 (Snap Scrolling) 效果，左右滑动切换状态列更加顺滑。

### 2. 仪表盘 (Dashboard)
- **置顶优先级**：快速查看被标记为“置顶”的关键任务。
- **今日及逾期**：聚焦于今天需要完成或已经逾期的任务。
- **等待我审核**：专门列出需要当前用户进行审核的任务。
- **委派给他人**：跟踪由我创建并委派给其他成员的任务。

### 3. 日历视图 (Calendar)
- **月历展示**：以直观的日历网格形式展示每月的任务完成情况。
- **热力图效**：根据每日完成的任务数量，日历格子会显示不同深浅的绿色（热力图），完成越多颜色越深。
- **每日活动时间轴**：点击特定日期，下方会弹出该日的详细活动时间轴，按小时展示任务的创建、状态变更、完成和删除等操作日志。
- **移动端适配**：针对手机屏幕优化了日历高度和时间轴的排版，确保底部导航栏不会遮挡内容。

### 4. 搜索与筛选 (Search & Filter)
- **全局搜索**：支持按标题和描述进行关键词搜索。
- **排序功能**：搜索结果自动按任务更新时间 (`updatedAt`) 从新到旧排列。
- **生成甘特图**：在搜索界面提供“生成甘特图”按钮。当填写了筛选条件或搜索词后，点击此按钮可为当前筛选出的任务生成直观的甘特图（时间线），方便查看任务排期和进度。
- **高级筛选组件**：
    - **搜索式多选**：所有筛选下拉框均升级为支持搜索的多选组件 (MultiSelect)，方便在大量选项中快速查找并选择多个条件。
    - **筛选维度**：
        - **创建人**：筛选任务的创建者（支持多选、排除）。
        - **状态**：筛选任务当前状态（支持多选、排除）。
        - **优先级**：筛选任务优先级（支持多选、排除）。
        - **负责人**：筛选任务当前负责人，支持“未指派”（支持多选、排除）。
        - **提出人**：筛选任务提出人/审核人，支持“无提出人”（支持多选、排除）。
        - **委派状态**：筛选已委派或未委派的任务（支持多选、排除）。
        - **媒介标签**：筛选任务来源或沟通渠道，支持“无媒介”（支持多选、排除）。
        - **自定义字段**：支持按所有类型为“单选”或“多选”的自定义字段进行筛选（支持多选、排除）。
    - **范围筛选**：支持按时间段（开始/截止）和进度范围（0-100%）筛选。
- **批量操作 (Batch Operations)**：
    - **批量选择**：
        - 支持点击任务卡片左上角的复选框选择单个任务。
        - 支持“全选/取消全选”按钮一键操作当前搜索结果中的所有任务。
    - **批量修改**：
        - **状态/优先级**：选中任务后，底部出现操作栏，可快速批量修改任务状态和优先级。
        - **批量编辑**：提供统一的编辑模态框，支持批量修改任务的状态、优先级、负责人、截止日期以及所有自定义字段。
            - 模态框中仅展示勾选的字段进行修改，未勾选的字段保持原值。
    - **批量删除**：支持一键删除所有选中的任务，删除前需二次确认。

### 5. 备忘录 (Memos)
- **独立记录**：提供单独的备忘录标签页，用于记录与任务无关的想法、灵感或笔记。
- **CRUD 操作**：支持备忘录的创建、修改、删除，并支持 inline 确认删除操作。

### 6. 任务详情与编辑 (Task Details)
- **移动端全屏弹窗**：在手机端，任务详情弹窗自动全屏显示，最大化利用屏幕空间；顶部操作按钮支持横向滑动，防止拥挤。
- **用户显示优化**：在所有涉及人员显示的地方（如任务卡片、详情页、筛选器），均采用“姓名 (单位)”的格式，以便在多单位协作场景下准确区分同名人员。
- **标签页布局**：任务详情页面采用标签页形式，分为“详情”、“活动日志”和“关联图谱”。
- **详情页**：包含任务标题、流程可视化、描述、子任务、关联任务以及任务字段侧边栏。
- **快速复制 (Quick Copy)**：在详情页右上角删除按钮左侧提供快速复制按钮。点击后将创建一个当前任务的副本，原样保留任务标题、描述、优先级、负责人、审核人、子任务及关联关系等所有信息，但会将新任务的“开始日期”重置为点击复制时的当天时间，并清空“截止日期”。
- **活动日志页**：专门展示活动日志，支持日志的查看与管理（删除、编辑）。
- **关联图谱页**：可视化展示任务间的关联关系，支持节点点击跳转、滚轮缩放及拖拽平移。
    - **可视化升级**：
        - 当前任务节点显示为紫色，且尺寸放大（1.5倍）。
        - 关联任务节点根据状态显示不同颜色：待办（浅红色）、进行中（浅黄色）、审核中（浅蓝色）、已完成（浅绿色）、延期（浅灰色）。
        - **图例说明**：在图谱右下角增加图例，清晰展示不同颜色代表的任务状态。
- **流程可视化**：直观展示任务从“待办”到“完成”的流转路径，特别标注委派状态。当任务处于“已完成”状态时，最后一个“完成”节点显示为绿色的已勾选节点。
- **任务冻结与解冻 (Freeze/Pause)**：
    - **暂停按钮**：在“流程可视化”区域的绝对右上角提供一个固定的**暂停键按钮**。
    - **冻结操作**：点击暂停按钮后，任务进入“已暂缓（冻结）”状态，系统会自动记录当前流程阶段。
    - **解冻操作**：冻结状态下，流程图中央会显示“已暂缓（冻结） - 点击解冻”按钮，点击即可一键恢复至冻结前的原始状态。
    - **日志记录**：冻结与解冻操作均会自动记录在任务的活动日志中。
- **子任务管理**：
    - 支持创建、勾选、修改子任务标题，细化工作内容。
    - 支持快速删除子任务。
    - **转为独立任务**：支持将子任务一键转换为独立任务，转换后自动保留与原父任务的关联关系。
- **关联任务**：支持任务间的双向关联，点击关联任务可直接跳转至目标任务详情。
- **侧边栏属性**：快速修改负责人、委派对象、审核人、优先级、日期、进度及标签。
- **优化人员选择**：负责人和委派对象字段优化，固定“我”在首位，其他人员隐藏并支持搜索选择。
    - **快捷操作**：在搜索人员时，若有匹配结果，按下回车键（Enter）可直接选中第一个匹配项。
- **删除确认**：所有删除操作（任务、备忘录、设置项）均采用页面内确认框，提升可见性。

### 7. 任务卡片展示 (Task Card)
- **视觉区分**：
    - **置顶任务**：显示红色边框和浅红色背景。
    - **委派任务**：显示蓝色边框和浅蓝色背景，并在右下角显示紫色委派标记。
- **简洁操作**：移除了卡片上的委派功能按钮，委派操作仅在任务详情页进行。

### 8. 快速捕获 (Quick Capture)
- **即时创建**：在页面顶部提供输入框，支持快速添加新任务，样式与各标签页保持一致。取消了吸顶效果，避免在移动端滚动时遮挡页面内容。
- **立即完成**：支持“立即完成”模式，适用于那些随手完成并需要记录的小事。
- **按需显示**：在“搜索”页面隐藏快速捕获栏，以便用户专注于搜索和批量操作；在其他页面保持显示。

### 9. 数据管理 (Data Management)
- **实时持久化**：所有操作实时保存至浏览器的 Local Storage，刷新页面不丢失数据。
- **导出数据**：支持将所有任务及设置导出为 JSON 文件进行备份。
- **导出 Excel**：支持将任务列表导出为 Excel 文件。
- **导入数据**：支持从 JSON 文件恢复数据，导入前提供覆盖确认提示，并显示操作反馈。

### 10. 系统设置 (Settings)
- **人员与组织管理**：
    - **团队管理**：自定义团队成员名单，支持编辑成员姓名、关联单位及职位。
    - **单位管理 (Entities)**：维护组织架构中的单位/部门列表，支持添加、编辑和删除。
    - **职位管理 (Positions)**：维护职位列表，支持添加、编辑和删除。
    - **关联设置**：在编辑成员时，支持通过多选组件 (MultiSelect) 快速关联多个单位和职位。
- **看板定制**：支持添加、修改或删除看板列，自定义列颜色和图标。
- **优先级定制**：自定义任务优先级及其对应的视觉样式。
- **媒介标签定制**：自定义任务来源或沟通渠道标签（如：微信、钉钉、邮件等）。
- **自定义字段**：支持添加文本、数字、日期、单选、多选字段，支持字段排序、显示隐藏，以及选项重排序。当选项超过10个时，自动转为搜索选择框。

## 技术特性
- **响应式设计**：深度适配桌面端和移动端。移动端采用底部导航栏布局，并针对各视图（看板滑动对齐、弹窗全屏化、安全区内边距等）进行了精细化适配，提供原生应用般的体验。
- **UI 穿透渲染**：所有多选下拉组件 (MultiSelect) 均采用 React Portal 技术实现，确保下拉菜单能够超出父容器边界显示，解决在表格底端或狭窄容器中选项被遮挡的问题。
- **图标系统**：集成 Lucide React 图标库，支持 Emoji 自定义图标。
- **状态管理**：基于 Zustand 实现高效的前端状态管理。
- **持久化方案**：使用 Zustand Persist 中间件。
